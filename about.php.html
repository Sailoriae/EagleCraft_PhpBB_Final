A Propos

Minecraft en Bref

Minecraft est un jeu vidéo de type « Bac à sable », créé par Markus « Notch » Person, puis développé par Mojang. Le joueur est plongé dans un monde cubique généré aléatoirement, qui simule une nature vierge et hostile. Il doit y survivre en exploitant les ressources naturelles, ou est libre de partir en exploration, ou bien de construire avec son imagination pour seul limite.

En effet, le jeu n'impose aucun objectif précis. Cette liberté donnée aux joueurs a conquis un large publique, créant une vaste communauté de fans, de créateurs et de développeurs, qui enrichissent le jeu. Cet enthousiasme a mené au portage du jeu sur d'autres plateformes avec la « Bedrock Edition ».

La version orignale, nommée « Java Edition », permet une grande modularité dans le système de jeu, d'où la multitude de serveurs proposant un jeu tourné vers le jeu de rôle (RolePlay) avec des sytèmes de villes et d'économie, le combat (PvP), ou encore les mini-jeux. De nombreuses variantes existent, et parmis elles, il y a celles proposées par EagleCraft.

Java Edition ? Bedrock Edition ?

Minecraft Java Edition est la version originale de Minecraft, disponible pour ordinateurs, nécéssitant Java sous Windows, Mac ou Linux. Vous devez l'avoir préalablement installé pour jouer sur EagleCraft. Rendez-vous sur Minecraft.net pour plus d'informations.

Minecraft, ou Bedrock Edition, est l'ensemble des versions pour smartphones (Anciennement « Pocket Edition ») et consoles, sur lesquelles les joueurs peuvent se rejoindre depuis la mise à jour « Better Together ». En revanche, ils ne peuvent pas jouer sur les serveurs Java Edition.

Un serveur « à l'ancienne »

Durant les premières années 2010, les serveurs Minecraft étaient des petites communautés chaleureuses avec un l'esprit d'être entre amis. C'est cet esprit amical que nous voulons conserver ici. De plus, ces serveurs proposaient juste de jouer au but original du jeu : Survivre, récolter des ressources, et surtout construire. C'est ce mode de jeu, en un peu plus RolePlay, qui est aujourd'hui le principal d'EagleCraft, dans les mondes Semi-RP.
