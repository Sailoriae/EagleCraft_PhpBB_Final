Bonjour à tous,
Depuis sa création en août 2012, EagleCraft accepte les versions crackées de Minecraft, c'est à dire des joueurs qui n'ont pas acheté le jeu.
Ainsi, EagleCraft a toujours été un serveur en mode "offline", ce qui l'oppose aux serveurs en mode "online" qui n'acceptent pas les versions piratées.

Ce choix n'était pas vraiment réfléchi, il venait juste du fait que certains de nos amis n'avaient à l'époque pas acheté le jeu. Et comme EagleCraft est le fruit de deux serveurs entre amis, nous avons continué à les accepter.
Durant ces 8 dernières années, j'ai remis quelques fois ce choix en question, mais comme EagleCraft n'a jamais réellement grandit, je n'ai jamais vu de réel intérêt à ne plus accepter les version crackées. Bon, en vrai, c'était surtout de la flemme. Et quant à Auldeor, il s'en fichait complétement.

Cependant, j'ai récemment fini de rendre tous les plugins d'EagleCraft compatibles avec le système d'UUID de Mojang. Si vous ne savez pas ce que c'est qu'un UUID, en gros, c'est l'identifiant uniquement et immodifiable de votre compte Minecraft.net. Mojang ont mis ce système en place afin de permettre aux joueurs de changer de pseudonyme !

Ainsi, nous avons fait le choix de passer EagleCraft en mode "online". Cependant, si vous ne le savez pas, un serveur "offline" n'a pas les UUID de ses joueurs synchronisés avec ceux de Mojang. Il faut donc modifier toutes les données du serveur pour les associer.
J'ai donc codé un script qui a convertit tous les UUID, donc vos inventaires, statistiques, résidences, balance d'Eagles, appartenance à des Villes, etc... Si votre pseudonyme sur EagleCraft correspondait à votre pseudonyme de votre compte Minecraft.net acheté, vous ne verrez aucun changement. Sinon, contactez-moi pour que je puisse faire quelque chose pour vous !

Cela implique donc que les joueurs n'ayant pas acheté Minecraft Java Edition ne pourront plus se connecter à EagleCraft. Certes, c'est dommage pour les personnes ne pouvant pas se le payer, mais je pense aujourd'hui qu'en tant que serveur publique, c'est mieux de respecter la licence Minecraft (EULA).
Si vous souhaitez l'acheter (Il coute 24 euros), rendez-vous sur cette page.

Bref, pour résumer :
  • EagleCraft n'accepte plus les versions crackées de Minecraft. Vous devez donc utiliser soit le Launcher officiel, soit de manière générale un Launcher utilisant votre compte Minecraft.net.
  • Si vous avez des problèmes, notamment de pseudonyme, contactez moi ! Sur le Discord d'EagleCraft par exemple.

A noter aussi qu'au passage, afin d'éviter des problèmes, j'ai modifié la position de déconnexion de tous les joueurs qui se sont déjà connecté au Spawn. Ainsi, la position de votre prochaine connexion (Et uniquement de la prochaine) sera forcément le Spawn, même si votre UUID a été convertit.

Voilà voilà... Bon jeu, et je ne sais pas, bon app' si vous allez passer à table !


PS : C'est donc la fin des commandes /login et /register. Oubliez votre mot de passe en jeu, il ne servira plus.
PPS : Si un jour Microsoft décident de tuer Minecraft Java Edition, nous pourrons faire l'opération inverse (Avec les MDPs du Forum), j'ai des scripts maintenant !
PPPS : Ouai on a toujours de vielles émoticônes, et l'impossibilité de mettre des emojis.
