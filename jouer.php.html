EagleCraft Jouer

Pour jouer sur EagleCraft, vous devez avoir acheté et installé Minecraft Java Edition. Pour ce faire, rendez-vous sur Minecraft.net.


Puis lancez le jeu en version 1.18.2.

Rendez-vous dans le menu Multijoueurs, et ajoutez un Nouveau Serveur...


Nom du Serveur :

EagleCraft

Adresse :

play.eaglecraft.fr

Connectez-vous à EagleCraft...

Bon jeu !

Vous commencez sur EagleCraft en tant que Vagabond ! Passez donc rapidement Villageois sur le Forum en vous Inscrivant et en publiant votre Présentation...


Vous pouvez aussi visiter le Codex où vous trouverez les guides et les règles du serveur Minecraft. Il est très recommandé de lire la page Guide des Nouveaux Joueurs pour bien comprendre les principes de base du serveur !
